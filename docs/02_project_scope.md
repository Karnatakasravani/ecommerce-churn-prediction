\# Project Scope



\## In-Scope Activities

\- Data ingestion from Online Retail dataset

\- Data cleaning and preprocessing

\- Feature engineering

\- Churn definition and labeling

\- Model training and evaluation

\- Streamlit web application for predictions

\- Batch prediction support

\- Documentation \& deployment



\## Out-of-Scope

\- Real-time production deployment

\- User authentication

\- Payment / CRM integration

\- Multi-language UX

\- Long-term MLOps automation



\## Users

\- Marketing analysts

\- CRM managers

\- Leadership team

\- Data \& business analysts



\## Deliverables

\- Trained ML model

\- Churn prediction API functions

\- Streamlit application

\- Evaluation metrics

\- Documentation

\- Business impact report



\## Constraints

\- Historical purchase data only

\- No demographic data available

\- No explicit feedback labels

\- Data is transactional, not session-based



\## Risks

\- Concept drift

\- Data imbalance

\- Churn definition uncertainty

\- Noise and missing data
